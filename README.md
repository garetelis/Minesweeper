# Minesweeper
This is a file to run in VS Code or any other Java compiler that allows you to solve randomly generated mazes in the terminal.

Try it out! 

With code: https://codehs.com/sandbox/garrettellis/minesweeper

Without code: https://codehs.com/sandbox/garrettellis/minesweeper/run

This demo runs in your browser so no need to install anything fancy, just click in and play!

You will be prompted for the size of board you want to solve and then will be able to pick spaces with the coordinates, place flags, and weed out all of the mines to win.

If you would like to download the file and run it elsewhere, just make sure to have the latest version of Java installed as the code uses enhanced switch-case blocks that are not compatible with previous versions.

If using the .zip file in the release just download, unzip, and run directly from the file or else it will not work.

You can also of course just download the .java file and run it directly from VS Code.
